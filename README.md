# popuration_transition

## description

人口推移を示したアプリケーションです
![demo](https://user-images.githubusercontent.com/43603215/64488309-62b2a600-d281-11e9-8ca8-f47edce4833b.gif)

## usage

`$ touch .env`

```:.env
RESAS_API_KEY=[YOUR_RESAS_API_KEY]

```

`$ yarn install`

// start dev server
`$ yarn dev`

## script

```
// start dev server
$ yarn dev

// build app
$ yarn build

// run prettier
$ yarn format

// run lint
$ yarn lint
```
