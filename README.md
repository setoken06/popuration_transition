# popuration_transition

## description

人口推移を示したアプリケーションです

## usage

`\$ touch .env`

```:.env
RESAS_API_KEY=[YOUR_RESAS_API_KEY]

```

`\$ yarn install`

// start dev server
`\$ yarn dev`

## script

```
// start dev server
$ yarn dev

// build app
$ yarn build

// run prettier
$ yarn format

// run lint
$ yarn lint
```
